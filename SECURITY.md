# Security Policy

If you discover any issues or bugs in ARES, please report them by creating an issue ticket on GitHub with the "bug" label. This will help us track and prioritize the problem.

[Bug_Report](https://github.com/AndraeCarotta/ARES/issues/new?template=bug_report.md)

## Reporting a Vulnerability

We only accept reports for the latest stable and official releases of ARES, or for versions currently in development or beta testing. You can find the latest release on our GitHub release page.

Reports for forks of ARES will not be accepted.
